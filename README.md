# Pre-Processamento_Dados_-_Pre_modelagem_-_EBAC
Projeto de pré-modelagem da EBAC que analisa dados de telecom para identificar quais serviços impactam o Monthly_Payment e quais variáveis influenciam o churn. Inclui EDA, boxplots interativos com Plotly e insights sobre serviços e fatores que afetam permanência e custos dos clientes.
